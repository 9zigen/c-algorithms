# c-algorithms

Usage:

```
mkdir build
cd build
cmake ..
make -j4
make test
```

How to make a coverage report:

```
mkdir build
cd build
../cov.sh
```

List of data structures and algorithms:

* Single- and double-linked lists (ilist.c)
* Red-black trees (rbtree.c)
* Hash tables (htable.c)
* MD5 Hashing (crypto/md5.c)
